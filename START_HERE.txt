Thank you for downloading the Endurance Race Stint Tracker!

This is a complete web application for tracking driver stint times with an
intelligent optimal stint calculator that learns from your performance.


┌───────────────────────────────────────────────────────────────────────────┐
│ 🚀 QUICK START (3 COMMANDS)                                               │
└───────────────────────────────────────────────────────────────────────────┘

  1. Make sure Node.js is installed (download from: https://nodejs.org)
  
  2. Open terminal/command prompt in this folder and run:
     
     npm install
     npm start
  
  3. Open your browser to: http://localhost:3000

│ ✨ WHAT DOES THIS APP DO?                                                 │
└───────────────────────────────────────────────────────────────────────────┘

  Core Features:
  ✓ Track driver stint times in real-time
  ✓ Color-coded warnings (green → orange → red)
  ✓ Automatic stint recording on driver changes
  ✓ Complete race history tracking
  ✓ Configurable race settings

  🆕 Optimal Stint Calculator:
  ✓ Learns from your completed stints
  ✓ Calculates ideal stint time (60% performance + 40% race needs)
  ✓ Provides strategic recommendations
  ✓ Shows confidence level and analytics
  ✓ Adapts as race progresses

┌───────────────────────────────────────────────────────────────────────────┐
│ 🆘 TROUBLESHOOTING                                                        │
└───────────────────────────────────────────────────────────────────────────┘

  Problem: "npm is not recognized" or "command not found"
  Solution: Install Node.js from https://nodejs.org
           Restart your terminal after installing

  Problem: Port 3000 already in use
  Solution: Stop other applications using that port
           Or edit server.js and change the PORT number

  Problem: Can't connect in browser
  Solution: Make sure server is running (check terminal)
           Try http://127.0.0.1:3000 instead
           Check firewall settings

  For detailed troubleshooting → See INSTALL.md
